# Assets Folder — What Goes Here

## 📸 Headshot
**File:** `headshot.jpg`
- Recommended size: 600 × 700 px
- Format: JPG (quality 85%) or WebP
- Style: Professional, well-lit, solid or blurred background
- Tip: A simple Studio-style shot on a white/grey backdrop works best

## 📄 Resume
**File:** `resume.pdf`
- Keep under 2MB
- Ensure it's text-selectable (not a scanned image)
- Filename must match exactly: resume.pdf

## 🖼️ Project Screenshots
| File | Section | Recommended Size |
|------|---------|------------------|
| `project-attrition.jpg` | Attrition Prediction Model | 800 × 500 px |
| `project-engagement.jpg` | Engagement Dashboard | 800 × 500 px |
| `project-orgdesign.jpg` | Retail Workforce Restructuring | 800 × 500 px |
| `project-thesis.jpg` | NLP Churn Prediction | 800 × 500 px |

**Tips for screenshots:**
- Use light-mode screenshots for consistency
- Crop to show the most compelling visual (a chart, dashboard, or key slide)
- If the project is confidential, use a blurred or anonymized version
- PowerPoint mockups work great if you don't have a real screenshot

## Notes
- All images should be optimized (use squoosh.app or imageoptim)
- If an image is missing, the site shows a tasteful placeholder automatically
